PhilAI - Philippine Legal AI Assistant
PhilAI is an AI-powered platform designed to provide accessible information about Philippine laws and jurisprudence. By leveraging the power of the Gemini API and a modern MERN-style architecture, it helps users navigate common legal offenses, understand their rights, and see how modern actions compare to established laws.

Purpose & Vision
The legal landscape in the Philippines can be difficult for many to navigate. PhilAI was built to:

Democratize Legal Knowledge: Make information about the Revised Penal Code and Special Penal Laws accessible to everyone.

Provide Contextual Clarity: Offer comparisons between everyday actions and legal precedents.

Support Transparency: Align with initiatives like TransparaBrgy to foster a more informed and legally aware community.

Key Features
Gemini-Powered Insights: Uses system-level instructions to ensure responses stay strictly within the context of Philippine law.

Taglish Support: Understands and responds in common Philippine linguistic nuances for better accessibility.

Secure Architecture: Backend-hosted API keys on Render and a high-speed frontend on Vercel.

Glassmorphism UI: A sleek, dark-themed interface built for focus and ease of use.

Tech Stack
Frontend: React (Vite), CSS-in-JS, Firebase SDK

Backend: Node.js, Express, CORS

AI: Google Gemini 2.5 Flash

Deployment: Vercel (Front) & Render (Back)

Disclaimer: PhilAI is for informational purposes only and does not provide professional legal advice. Always consult with a licensed attorney for legal matters.
